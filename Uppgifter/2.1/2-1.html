2.1 Grunderna i CSS 3

Externt länkad CSS-fil och dess fördelar
Det finns tre sätt att implementera CSS: External Style Sheet, Internal Style Sheet ovh Inline Style. Fördelen med den förstnämda är att man kan använda samma fil för att ändra utseende på alla sidor som använder filen samtidigt, istället för att ändra varje sida var för sig. Detta underlättar då man vill ha ett enhetligt utseende på ett projekt med flera sidor.

Arv
Arv läggs till på en property för att sätta värdet till samma värde som elementet det ligger i har. Default värdet på olika properties varierar.

/*background: Före: transparent. Efter: white*/
background: white;

/*color: Före: Samma som parent. Efter: blue*/
color: blue;

/*margin: Före: 0 Efter: Samma som parent*/ margin: inherit;

Cascade
Cascade är en algoritm som avgör vilken CSS som skall användas på en property då flera värden har blivit angivna.

Selektorer (tagg-, attribut-, klass-, id- och barn-selektorer)
Det finns olika sätt att definiera en selektor. Nedan återfinns några av dem.

/*Tagg: Sätter CSS på alla h1 element*/
h1 {
color: red; }

/*Attribut: Sätter CSS på alla element som har attributet target=su*/
[target=su] {
background: red;
}

/*Klass: Sätter CSS på alla element med klass-namnet "myClass"*/
.myClass {
color: red;
}

/*id: Sätter CSS på elementet med id=myId*/
#myId {
color: red;
}

/*barnelement: Sätter CSS på alla h1 som har en header som parent*/
header > h1{
color: green;
}

/*barnelement: Sätter CSS på alla h1 inuti en header*/
header h1 {
color: green;
}

Enheter (absoluta: in, cm, mm, pt, pc, px och relativa: em, ex, %)
Enheter kan delas in absoluta och relativa storlekar. De absoluta behåller samma storlek medan de relativa beror på hur stort browserfönstret är eller storleken på dess parent-element.

/*ABSOLUTA ENHETER*/

/*in: Sätter höjden på divar till 1 tum*/

div {
height: 1in;
}

/*cm: Sätter höjden på divar till 1 centimeter*/
div {
height: 1cm;
}

/*mm: Sätter höjden på divar till 1 millimeter*/
div {
height: 1mm;
}

/*pt: Sätter höjden på divar till 1 point (1 point = (1/72)tum))*/
div {
height: 1pt;
}
/*pc: Sätter höjden på divar till 1 picas (1 picas = (1/12)tum)*/
div {
height: 1pc;
}

/*px: Sätter höjden på divar till 1 bildpunkt (1 bildpunkt = (1/96)tum)*/ /*skärmar med hög upplösning har fler bildpunkter definierad som 1 pixel*/
div {
height: 1px;
}

/*RELATIVA ENHETER*/
/*em: Sätter höjden på divar till samma storlek som storleken på typsnittet för det aktuella elementet*/
div {
height: 1em;
}

/*ex: Sätter höjden på divar till storleken på hur hög ett gemen "x" är för typsnittet för det aktuella elementet*/
div{
height: 1ex;
}
/*%: Sätter höjden på divar till 1% av hur högt dess parent-element är*/
div {
height: 1%;
}

Hur ett CSS-dokument är uppbyggt
Först i dokumentet skall eventuell teckenuppsättning anges, varefter importering av exempelvis andra CSS-dokument eller fonts ligger. Detta kan göras antingen genom att ange en string eller en url. Sedan kommer selektorn som avgör vilket element som skall påverkas. Inom måsvingarna kan man ange alla de egenskaper man vill sätta ett värde på.
