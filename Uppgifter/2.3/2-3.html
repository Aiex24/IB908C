2.3 Marginaler, kantlinjer och paddning

Boxmodell
Varje element kan ha ett innehåll som har en kantlinje omkring sig. Området mellan kantlinjen och innehåll kallas padding, och området utanför kantlinjen till nästa element kallas för margin.

border-color: sätter färgen på kantlinjen
border-style: Här kan man sätta utseendet på kantlinjen, exempelvis solid, dubbel linje, prickar med mera
border-width: sätter tjockleken på kantlinjen
border: kan användas för att sätta värden på alla ovanstående properties på samma rad
outline: skillnaden mellan outline och border är att outline inte tar upp något utrymme. Ett användningsområde är att lägga in outlines vid debugging för att lättare se var olika element slutar


div med en kantlinje i svart och outline i rött

visibility: Kan bland annat sättas till visible (elementet är synligt), hidden (elementet synns inte men tar upp plats), collapse (elementet varken syns eller tar upp plats) om elementet är av typen table annars fungerar collapse på samma sätt som hidden. För att få collapse-effekt på övriga element används display: none.
clip-path: Definierar ett område av ett element som skall vara synligt. Exempelvis kan man definiera en cirkel, ellipse eller polygon.
display: avgör hur ett element skall visas upp (renderas). Några vanliga värden är inline, block, inline-block, flex, och table.
